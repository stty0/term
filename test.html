SSH Client 테스트 결과

✅ 백엔드 서버: http://localhost:8000 - 정상 동작
✅ 프론트엔드 서버: http://localhost:3000 - 정상 동작
✅ WebSocket 연결: 정상 동작
✅ SSH 연결: 192.168.10.100:22 - 정상 동작
✅ 명령어 실행: whoami 명령어 실행 성공 (결과: jrpark)
프로젝트 구조: 
term/
├── backend/                 # FastAPI + WebSocket + SSH
│   ├── main.py             # 메인 서버 파일
│   └── requirements.txt    # Python 의존성
├── frontend/               # Vue.js + TypeScript + xterm.js
│   ├── src/
│   │   ├── App.vue        # 메인 Vue 컴포넌트
│   │   ├── main.ts        # 엔트리 포인트
│   │   └── style.css      # 스타일시트
│   ├── package.json       # Node.js 의존성
│   └── index.html         # HTML 템플릿
└── README.md              # 프로젝트 설명
사용 방법:
  1. 백엔드 서버 실행: cd backend && python main.py
  2. 프론트엔드 서버 실행: cd frontend && npm run dev
  3. 브라우저에서 http://localhost:3000 접속
  4. SSH 로그인 정보 입력 후 연결
  5. 터미널에서 명령어 실행
주요 기능:
  • 웹 기반 SSH 클라이언트 (PuTTY 대체)
  • xterm.js를 사용한 실제 터미널 UI
  • 다중 SSH 세션 지원 (탭 기능)
  • 실시간 WebSocket 통신
  • ANSI 색상 코드 지원
